Muitas operações de negócios giram em torno do rastreamento de quando os suprimentos estão baixos, quando são substituídos e a quais dispositivos pertencem. A página **Gerenciar suprimentos** pode ajudá-lo a ser proativo em relação ao gerenciamento de suprimentos, entendendo as principais métricas, como quando um suprimento será substituído, a taxa de preenchimento do suprimento ou o número de páginas perdidas devido a substituições prematuras de suprimentos.

## Visualizando suprimentos {#viewing-supplies}
Você pode exibir suprimentos para todos os dispositivos em uma conta ou para um dispositivo específico. O Print Tracker fornece os suprimentos atualmente instalados para cada dispositivo, bem como o histórico de suprimentos para cada dispositivo. Isso pode ajudá-lo a obter uma visão abrangente de como seus clientes consomem seus suprimentos.

### Suprimentos por Conta {#supplies-by-entity}
![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-viewing-supplies-by-entity.gif)

Para exibir suprimentos por conta:
1. Certifique-se de que selecionou a conta para a qual pretende ver os suprimentos
2. Usando a barra lateral, clique em **Dispositivos > ver suprimentos**

### Suprimentos por dispositivo {#supplies-by-device}
![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-viewing-supplies-by-device.gif)

Para exibir suprimentos para um dispositivo específico:
1. Usando a barra lateral, clique em **Dispositivos > Visualizar dispositivos**
2. Clique no dispositivo para o qual você deseja visualizar os suprimentos
3. Clique na guia **Suprimentos**


### Personalizando colunas {#customizing-columns}
![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-changing-columns.gif)

Você pode personalizar quais colunas aparecem na tabela de suprimentos. Isso geralmente é útil se você estiver usando o Print Tracker em uma tela pequena ou se estiver interessado apenas em algumas das colunas.

Para personalizar colunas:
1. Clique no ícone **Filtro** no canto superior direito da tabela
2. Verifique as colunas que você deseja que apareçam na tabela

### Incluindo Filhas {#including-children}
![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-include-children.gif)

Opcionalmente, você pode incluir suprimentos que pertencem a dispositivos e contas que estão abaixo da conta atual em sua [hierarquia](./entities#hierarchy). Isso é chamado de "incluir filhas" e pode ser feito em praticamente qualquer lugar no Print Tracker, inclusive na página de gerenciamento de suprimentos. 

Para incluir filhas:
1. No canto superior direito da tabela, clique no botão de alternância **Incluir filhas**

### Incluindo suprimentos substituídos {#including-replaced-supplies}
![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-include-replaced.gif)

Por padrão, apenas os suprimentos instalados atualmente são mostrados, no entanto, você também pode querer ver suprimentos que foram instalados anteriormente em um dispositivo, mas que já foram substituídos. Você pode fazer isso incluindo os suprimentos substituídos.

Para incluir suprimentos substituídos:
1. No canto superior direito da tabela, clique no ícone **Filtro**
2. Marque a caixa de seleção **Include replaced supplies**
3. Um novo filtro de intervalo de datas aparecerá na área de filtro de tabela, este seletor de datas permite que você escolha um intervalo de datas para o qual você deseja ver suprimentos substituídos. Por padrão, esse intervalo de datas é definido como os últimos 30 dias.

### Filtrando e classificando {#filtering-and-sorting}
![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-filter-sort.gif)

Você pode filtrar a lista de suprimentos digitando um termo de pesquisa na barra de pesquisa na parte superior da tabela. Você pode pesquisar dispositivos específicos por número de série, por modelo, nome de suprimento e muito mais.

Você também pode classificar a lista de suprimentos clicando nos cabeçalhos das colunas. Clique nos cabeçalhos das colunas várias vezes para ajustar a direção de classificação (ascendente ou descendente).

### Paginação {#pagination}
![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-pagination.gif)

Contas com muitos dispositivos podem ter centenas ou milhares de suprimentos. Para ajudá-lo a encontrar os dados que você está procurando, o Print Tracker divide todos os seus dados em páginas. Você pode navegar entre páginas usando os controles de paginação na parte inferior da tabela.

### Exportando {#exporting}
![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-export-to-csv.gif)

Você pode exportar a lista de suprimentos para um arquivo CSV. Isso é útil se você quiser compartilhar a lista de suprimentos com outra pessoa ou se quiser fazer alguma análise em uma planilha.

### Páginas Perdidas {#lost-pages}

Um fabricante de suprimento pode fornecer uma classificação de rendimento para um suprimento. Este é o número de páginas que o suprimento deve imprimir. Se um suprimento for substituído antes do tempo estimado, o número de páginas que o suprimento deveria imprimir será perdido. Isso é chamado de "páginas perdidas". As páginas perdidas podem ajudá-lo a identificar quando seus clientes estão substituindo suprimentos muito cedo. Para que o Print Tracker calcule as páginas perdidas, ele precisa saber o rendimento nominal do suprimento. Em muitos casos, o rendimento esperado é fornecido pelo dispositivo, no entanto, se o dispositivo não fornecer o rendimento nominal, então você pode [inserir manualmente o rendimento esperado](#rated-yield) para cada suprimento.

Se a coluna "Páginas perdidas" estiver ausente da tabela de suprimentos, você poderá adicioná-la [customizando as colunas](#customizing-columns).

### Taxa de preenchimento {#fill-rate}

A taxa de preenchimento (muitas vezes referida como taxa de cobertura) é a porcentagem média da página que é coberta pelo suprimento. Normalmente, um fabricante de suprimentos definirá o [rendimento nominal](#rated-yield) de um suprimento em alguma [taxa de preenchimento esperada](#expected-fill-rate). A maioria dos suprimentos tem um rendimento nominal com uma taxa de preenchimento de 5%. Isso significa que um suprimento pode não imprimir o rendimento nominal se um cliente tiver uma taxa de preenchimento mais alta, indicando que o cliente está colocando mais suprimento na mesma página de tamanho do que o suprimento foi originalmente classificado.

A taxa de preenchimento pode ajudá-lo a entender a eficiência com que um suprimento está sendo utilizado e determinar o custo geral por página. Ao comparar as taxas de preenchimento de diferentes suprimentos, pode-se identificar a opção mais econômica para suas necessidades de impressão. Além disso, o monitoramento da taxa de preenchimento pode ajudar a otimizar o processo de impressão, reduzir o desperdício e estender a vida útil dos suprimentos de impressão. Isso acaba levando à redução dos custos operacionais e à melhoria da sustentabilidade no longo prazo. A taxa de preenchimento também serve como referência para avaliar o desempenho de suprimentos em vários fabricantes, permitindo que empresas e consumidores tomem decisões baseadas em dados ao selecionar seus recursos de impressão.

### Data de esgotamento estimada <BetaLabel/> {#estimated-depletion-date-betalabel}

Com base em padrões históricos de impressão, o Print Tracker tentará prever quando um suprimento atingirá 0%. Isso é chamado de data estimada de esgotamento. Alguns dispositivos calcularão e relatarão essa métrica ao Print Tracker, no entanto, se o dispositivo não fornecer essa métrica, o Print Tracker tentará calculá-la com base nos padrões históricos de impressão do dispositivo.

## Gerenciando suprimentos {#managing-supplies}

Você pode definir algumas configurações dos seus suprimentos para ajudar o Print Tracker a fornecer melhores insights de suprimentos e para ajudar suas equipes de atendimento de suprimentos a serem mais proativas em relação ao gerenciamento de suprimentos.

### Número da peça {#part-number}
O número de peça de suprimento geralmente é útil para as equipes de gerenciamento de suprimentos, permitindo que elas solicitem novamente os mesmos números de peça que estão instalados atualmente em seus dispositivos. Em alguns casos, o Print Tracker será capaz de detectar automaticamente o número de peça de um suprimento.

Se saber o número da peça de suprimento for importante para você e ele não for coletado pelo Print Tracker, você poderá especificar manualmente o número de peça de um suprimento. Este número de peça será automaticamente aplicado a todos os suprimentos futuros do mesmo tipo. Por exemplo, se você especificar manualmente um número de peça para um toner preto e o toner preto for eventualmente substituído, o toner preto de substituição também assumirá o mesmo número de peça.

![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-set-part-number.gif)

Para especificar manualmente o número da peça de um suprimento:
1. Usando a barra lateral, clique em **Dispositivos > ver suprimentos**
2. Encontre o suprimento para o qual você deseja especificar o número da peça
3. Clique na célula para o suprimento que você deseja alterar, e na coluna **Número da peça**
4. Digite o número da peça para o suprimento
5. Pressione **Enter** para salvar o número da peça

### Taxa de preenchimento esperada {#expected-fill-rate}
A taxa de preenchimento (muitas vezes referida como taxa de cobertura) é a porcentagem média da página que é coberta pelo suprimento. Normalmente, um fabricante de suprimentos definirá o [rendimento nominal](#rated-yield) de um suprimento em uma taxa de preenchimento específica chamada **Taxa de Preenchimento Esperada**. A maioria dos suprimentos tem um rendimento nominal com uma taxa de preenchimento esperada de 5%. Isso significa que um suprimento pode não imprimir o rendimento nominal se um cliente tiver uma taxa de preenchimento maior do que a taxa de preenchimento esperada, indicando que o cliente está colocando mais suprimento na mesma página de tamanho do que o suprimento foi originalmente classificado.

A menos que especificado de outra forma, o Print Tracker assume que todos os suprimentos são classificados em uma taxa de preenchimento esperada de 5%. Se você souber a taxa de preenchimento esperada para um suprimento, poderá especificá-la manualmente. Esta taxa de preenchimento esperada será automaticamente aplicada a todos os suprimentos futuros do mesmo tipo. Por exemplo, se você especificar manualmente uma taxa de preenchimento esperada para um toner preto e o toner preto for eventualmente substituído, o toner preto de substituição também assumirá a mesma taxa de preenchimento esperada.

![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-set-expected-fill-rate.gif)

Para especificar manualmente a taxa de preenchimento esperada para um suprimento:
1. Usando a barra lateral, clique em **Dispositivos > ver suprimentos**
2. Encontre o suprimento para o qual você deseja especificar a taxa de preenchimento esperada
3. Clique na célula para o suprimento que você deseja alterar, e na coluna **Taxa de preenchimento esperada**
4. Digite a taxa de preenchimento esperada para o suprimento
5. Pressione **Enter** para salvar a taxa de preenchimento esperada

### Rendimento nominal {#rated-yield}
O rendimento nominal é determinado por um fabricante de suprimento e representa o número esperado de páginas que o suprimento deve imprimir, dadas algumas suposições padrão (como a [taxa de preenchimento esperada](#expected-fill-rate)). O rendimento nominal é usado para calcular métricas como [páginas perdidas](#lost-pages). Em muitos casos, ele é fornecido pelo dispositivo, no entanto, se o dispositivo não fornecer o rendimento nominal, você pode inserir manualmente o rendimento nominal para cada suprimento.

Este rendimento nominal será automaticamente aplicado a todos os suprimentos futuros do mesmo tipo. Por exemplo, se você especificar manualmente um rendimento nominal para um toner preto e o toner preto for eventualmente substituído, o toner preto de substituição também terá o mesmo rendimento nominal.


![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-set-rated-yield.gif)

Para especificar manualmente o rendimento nominal de um suprimento:
1. Usando a barra lateral, clique em **Dispositivos > ver suprimentos**
2. Encontre a suprimento para o qual você deseja especificar o rendimento nominal
3. Clique na célula para o suprimento que você deseja alterar, e na coluna **Rendimento nominal**
4. Digite o rendimento nominal para o suprimento
5. Pressione **Enter** para salvar o rendimento nominal

### Custo de Suprimento {#supply-cost}
O custo de suprimento é o preço que você paga por um suprimento. Isso é usado para calcular o custo por página de um suprimento. Infelizmente, esses dados não são fornecidos ao Print Tracker pelo dispositivo, portanto, você precisará inserir manualmente o custo de suprimento para cada suprimento se você se importar com qualquer uma das métricas baseadas em custo de suprimento.

Este custo de suprimento será automaticamente aplicado a todos os suprimentos futuros do mesmo tipo para este dispositivo. Por exemplo, se você especificar manualmente um custo de suprimento para um toner preto e o toner preto for eventualmente substituído, o toner preto de substituição também assumirá o mesmo custo de suprimento.

![](https://www.cdn.printtrackerpro.com/images/documentation/managing-supplies-set-supply-cost.gif)

Para especificar manualmente o custo de suprimento de um suprimento:
1. Usando a barra lateral, clique em **Dispositivos > gerenciar suprimentos**
2. Encontre o suprimento para o qual você deseja especificar o custo de suprimento
3. Clique na célula para o suprimento que você deseja alterar, e na coluna **Custo de fornecimento**
4. Digite o custo de suprimento para o suprimento
5. Pressione **Enter** para salvar o custo de suprimento
