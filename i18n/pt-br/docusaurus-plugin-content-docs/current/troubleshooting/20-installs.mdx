# Instalações

## Antivírus {#antivirus}
Em alguns casos, o software antivírus pode interferir na instalação e operação do Print Tracker. Embora isso seja incomum na maioria dos computadores, algumas soluções antivírus de nível empresarial podem sinalizar o Print Tracker como malware. É importante entender que o Print Tracker não é malware e que é seguro instalar e executar no seu computador. Às vezes, o antivírus assume falsamente que o Print Tracker é malware porque exibe muitos dos mesmos comportamentos que o malware real, por exemplo, o Print Tracker :

* Atualiza-se automaticamente quando uma nova versão está disponível.
* Inicia automaticamente quando o computador é iniciado.
* Verifica sua rede usando vários protocolos.
* Coleta dados de dispositivos em sua rede e carrega esses dados para um servidor.
* Faz chamadas de sistema de baixo nível para a API do Windows, a fim de falar com o spooler de impressora do Windows.
* Armazena e recupera valores do registro do Windows.
* Verifica a lista de processos para encontrar outras instâncias do Print Tracker que possam estar em execução.
* Lê e grava arquivos de log e banco de dados para o sistema local.

Embora existam semelhanças nas heurísticas do Print Tracker e do malware, o Print Tracker é um conhecido provedor de agentes de coleta de dados e está na indústria há 19 anos.

### Problemas de instalação {#installation-issues}
:::tip
Se você estiver tendo problemas para instalar o Print Tracker, tente usar um instalador genérico em vez de um instalador com um clique.
:::
O Print Tracker suporta [vários métodos diferentes](.. /guides/40-installing-agents.md#deployment-strategies) e alguns métodos são mais propensos à interferência antivírus do que outros. Por exemplo, o Print Tracker fornece instaladores de um clique que são mais propensos a serem sinalizados por software antivírus porque esses instaladores são gerados em tempo real e não são assinados usando nossos certificados de assinatura, enquanto nossos instaladores genéricos são assinados usando nossos certificados de assinatura e são menos propensos a serem sinalizados por software antivírus.

## Políticas de Rede {#network-policies}
A maioria das redes suporta todos os recursos do Print Tracker prontos para uso, sem modificações. Descobrimos que algumas redes mais restritivas podem exigir alguma configuração adicional para permitir que o Print Tracker se comunique com as impressoras em sua rede ou com nossos servidores.

Se você estiver enfrentando problemas para fazer com que o Print Tracker funcione corretamente, verifique se sua rede é compatível com os nossos [requisitos de rede](.. /system-requirements/overview.md#network-requirements).

## Laptops {#laptops}
Embora o Print Tracker possa ser instalado em muitas plataformas e hardwares diferentes, geralmente desaconselhamos a instalação do Print Tracker em laptops. Os laptops são frequentemente removidos da rede, são levados para casa à noite e nos fins de semana, ou são colocados em modo de suspensão. 

A precisão dos dados que o Print Tracker coleta depende de o agente estar on-line e ser capaz de se comunicar com as impressoras o máximo possível. Se o agente estiver offline, ele não poderá coletar dados das impressoras. Se o agente estiver offline por um longo período de tempo, ele pode perder informações críticas como substituições de suprimentos.
